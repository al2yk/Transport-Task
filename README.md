# Решение транспортных задач на C++

Проект реализует методы решения транспортных задач линейного программирования, включая построение начального опорного плана и его оптимизацию. Программа предоставляет консольный интерфейс для ввода данных, вычисления решения и отображения результатов.

## Начало работы

Эти инструкции помогут вам склонировать проект и запустить его на локальном компьютере для разработки и тестирования.

### Необходимые условия

Для сборки и запуска проекта на Windows вам потребуется:

- [Visual Studio Community](https://visualstudio.microsoft.com/ru/downloads/) (бесплатная версия)
- При установке выберите "Разработка настольных приложений на C++"
- CMake (обычно входит в состав Visual Studio)
- [GitBash](https://gitforwindows.org/) 

### Установка на Windows c использованием Visual Studio

1. **Клонируйте репозиторий** через командную строку или Git Bash:
```cmd
git clone https://github.com/al2yk/Transport-Task.git
cd Transport-Task
```

2. **Откройте проект в Visual Studio:**
   - Запустите Visual Studio
   - Выберите "Открыть локальную папку"
   - Укажите папку с проектом
   - Visual Studio автоматически обнаружит файл CMakeLists.txt

3. **Настройте сборку:**
   - В верхней панели выберите конфигурацию сборки (Debug или Release)
   - Выберите целевую систему (x64 или x86)

4. **Соберите проект:**
   - В меню выберите "Сборка" → "Собрать все"
   - Или нажмите F7

5. **Запустите программу:**
   - Нажмите Ctrl+F5 (запуск без отладки)
   - Или F5 (запуск с отладкой)

### Проверка установки

После успешной сборки запустите программу. Вы должны увидеть приглашение для ввода данных:

```
Введите количество поставщиков:
```

Пример ввода данных:
```
Введите количество поставщиков: 2
Введите количество потребителей: 3

Введите запасы поставщиков (через пробел): 300 400
Введите потребности потребителей (через пробел): 200 250 250

Введите матрицу стоимостей:
Строка 1: 5 8 6
Строка 2: 7 4 9
```
